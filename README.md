## Running

### Development

Make `config/constants.js` (see `config/constants.production.js` for format.)

Then run

    $ node bin/www

### Production

- Set the appropriate environment variables (see `config/constants.production.js`).
- Set `NODE_ENV` to `production`.
- Push/Deploy.
